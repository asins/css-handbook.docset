<image>

  • 版本：CSS3

语法：

<image> = <url> | image() | image-set() | element() | cross-fade() | <gradient>

说明：

<image> 值表示一个2D图像，可以是url引用，图像符号，渐变符号。
  • <image> 可以用在包括：border-image-source, background-image, list-style-image, cursor 等属性上，它被用来取代 <url> 属性值，因为 <image> 值已包含了 <url> 在内。
  • 有些场景下，图像可能是无效，比如当 <url> 指向的是一个无效的图像资源时。一个无效的图像将表现为全透明，但是当无效的图像需要一些特殊行为时，可以使用 image()
