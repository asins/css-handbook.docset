<percentage>

  • 版本：CSS2

说明：

<number>接%
  • 百分比总是相对于另外一个值才存在的。

    示例代码：

    div { font-size: 12px; } .test { font-size: 200%; }
  • 百分比的单位是：%

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
